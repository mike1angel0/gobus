- Learnings: Current codebase has no XSS vectors — React auto-escapes JSX, React Router `to` prop doesn't support javascript: scheme, Leaflet uses React components. The sanitizeUrl helper is preventive for when user-provided URLs (avatarUrl, logos) are added.
